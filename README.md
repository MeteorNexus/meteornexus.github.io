# MeteorNexus Blog

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://meteornexus.github.io)

这是 MeteorNexus 团队的官方技术博客站点源码。本项目基于广受好评的 [Hux Blog](https://github.com/Huxpro/huxpro.github.io) Jekyll 主题构建，并进行了深度定制，旨在分享与传播数值天气预报（NWP）领域的专业知识与工程实践。

## 🎯 博客使命

我们是一群来自全球、经验丰富的气象算法工程师与科学家。此博客是我们的知识分享中心，专注于：
- **核心技术**：WRF、WRFDA、GSI、WRF-Chem 等数值模式的应用、调试与优化。
- **实践指南**：分享从实际项目中提炼的教程、最佳实践与案例分析。
- **社区赋能**：帮助高校学生、研究人员及相关领域专业人士降低NWP学习门槛，更专注于科学问题本身。
